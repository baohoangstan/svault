### 4. Use the CLI

The `./cli.sh` script gives you powerful commands:

```bash
# Sync all changes to Git
./cli.sh sync

# View tasks across the vault
./cli.sh tasks

# Read a note with backlinks
./cli.sh read "note-name"

# Check git status
./cli.sh status
```

For Obsidian integration (requires Obsidian installed):
```bash
# Create daily note
./cli.sh daily

# Search notes
./cli.sh search

# Create new note
./cli.sh create -n "Note Name"
```
